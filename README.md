# WeightLattice Documentation

This package contains an updated Read the Docs starter documentation for the Blender addon **WeightLattice**.

## Included

- Full Sphinx + MyST documentation structure
- Read the Docs configuration
- Workflow pages
- Shortcut reference
- Preferences reference
- Troubleshooting and FAQ
- Updated automotive screenshots replacing the older monkey example

## Suggested usage

1. Copy the `docs/` folder into your addon repository.
2. Copy `.readthedocs.yaml` and `requirements.txt` to the root of the repository.
3. Push the repository to GitHub.
4. Import the project into Read the Docs.
5. Build the documentation.
