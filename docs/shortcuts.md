# Keyboard Shortcuts

All shortcuts are active **only while a WeightLattice session is running**. Otherwise they fall through to Blender's native keys.

## Weight Paint

| Key | Action |
|---|---|
| `Enter` | Create the lattice — or open it if it already exists |
| `U` | Refit the lattice bounds to the current weights |
| `Q` | Switch to Edit Lattice |
| `Shift+E` | Switch to Edit Mesh |
| `B` | Draw brush |
| `L` / `R` | Linear / Radial gradient |
| `I` / `X` / `S` | Invert / Delete / Smooth weights |
| `Ctrl+Scroll` | Adjust brush weight from 0 to 1 |
| `Alt+F` | Fill the active vertex group at weight 1.0 |
| `ESC` | Exit the session |

## Edit Lattice

| Key | Action |
|---|---|
| `Enter` | Apply the lattice |
| `ESC` | Remove the lattice without applying |
| `Q` | Return to Weight Paint |
| `Shift+E` | Switch to Edit Mesh |
| `Alt+C` | Lattice Properties popup |

## Edit Mesh

| Key | Action |
|---|---|
| `Shift+E` | Switch to Edit Lattice |
| `Q` | Return to Weight Paint |
| `Alt+C` | Edit Mesh Properties popup |

## Rebinding

Every shortcut can be rebound from **Edit > Preferences > Add-ons > WeightLattice > Shortcuts**, with live key capture, per-shortcut reset and a global reset. See [Preferences](preferences.md).
