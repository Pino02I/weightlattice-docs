# Shortcuts

WeightLattice includes shortcuts to make the workflow faster and more direct.

## Weight Paint mode

- **Enter**: Create or update the lattice.
- **Double-click Left Mouse**: Create or update the lattice.
- **B**: Activate the draw brush.
- **L**: Activate the linear gradient.
- **R**: Activate the radial gradient.
- **I**: Invert weights.
- **X**: Delete weights.
- **S**: Smooth weights.
- **U**: Refit lattice bounds.
- **W**: Go to Edit Lattice.
- **ESC**: Exit the modal workflow.

## Edit Lattice mode

- **Enter**: Apply the lattice.
- **Double-click Left Mouse**: Apply the lattice.
- **ESC**: Remove the lattice.
- **Double-click Right Mouse**: Remove the lattice.
- **W**: Return to Weight Paint.
- **X**: Open lattice properties.
- **Numpad +**: Increase lattice resolution on all axes.
- **Numpad -**: Decrease lattice resolution on all axes.

## Notes

The exact behavior depends on the active mode and on whether the current mesh already has a lattice linked to the active vertex group.
