# Quick Start

This section shows the shortest path from a mesh to a working WeightLattice setup.

## 1. Select a mesh

Start from a mesh object in Blender.

## 2. Create or choose a vertex group

WeightLattice works with the active vertex group. You can create a new one from the addon or use an existing group.

## 3. Paint the influence area

Switch to Weight Paint mode and paint the region that should drive the lattice deformation.

## 4. Create the lattice

Press **Enter** or click **Create / Update Lattice**.

## 5. Edit the lattice

Go to Edit Lattice mode and adjust the cage until the deformation matches your goal.

## 6. Finalize the result

When the deformation is ready, apply the lattice or remove it without applying.

## Result

You now have a repeatable workflow that converts painted weight influence into a deformation cage you can refine directly.
