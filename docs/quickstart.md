# Quick Start

This example shows the fastest way to use WeightLattice on a real modeling task.

## 1. Select the mesh

Start from the object you want to deform and open the **WeightLattice** panel in the 3D Viewport sidebar.

## 2. Create a WeightLattice group

Use **New Weight Lattice** to create a new vertex group and jump into Weight Paint mode.

## 3. Paint the target area

Paint the part of the model that should be controlled by the lattice. In this example, the painted area defines the front section of the car body.

## 4. Create the lattice

Press **Enter** or click **Create / Update Lattice** to generate a lattice around the painted region.

## 5. Edit the cage

Switch to Edit Lattice mode and reshape the cage to create the desired form.

## 6. Apply the result

When the new shape is correct, apply the lattice to transfer the deformation back to the mesh.

## Result

You can quickly prototype controlled shape variations without manually editing the full mesh topology.
