# Installation

## Requirements

WeightLattice requires:

- Blender 3.0 or newer.
- A standard Blender Python environment.
- The addon files distributed as a ZIP or Python package.

## Install the addon

1. Open Blender.
2. Go to **Edit > Preferences > Add-ons**.
3. Click **Install...**
4. Select the addon archive or Python file bundle.
5. Enable **WeightLattice** in the addon list.

## Confirm it is active

When the addon is enabled, a dedicated WeightLattice panel appears in the 3D Viewport sidebar in the appropriate mode.

## Update the addon

If you are replacing an older version:

1. Disable the previous version.
2. Install the new files.
3. Re-enable the addon.
4. Restart Blender if needed.

## Compatibility note

WeightLattice is built around Blender’s mesh, weight paint, and lattice systems, so the exact UI may differ slightly between Blender versions.
