# Installation

WeightLattice is packaged as a **Blender Extension** and requires **Blender 4.2 LTS or newer**.

## Drag and drop (fastest)

1. Open Blender.
2. Drag `WeightLattice_1.0.0.zip` from your file browser and drop it anywhere in the Blender window.
3. Click **OK** in the confirmation dialog.

## Install from Disk

1. Go to **Edit > Preferences > Get Extensions**.
2. Click the drop-down arrow in the top-right corner of the Preferences window and choose **Install from Disk...**.
3. Select `WeightLattice_1.0.0.zip`. The extension is installed and enabled automatically.

## Verify the installation

In **Edit > Preferences > Add-ons**, search for *WeightLattice* — it should be listed and enabled. In the 3D Viewport, press `N`: a **WeightLattice** tab appears in the sidebar.

```{note}
Do not unzip the file — Blender installs the `.zip` directly.
```

## Updating

Install the new version over the old one — no need to uninstall first.

## Uninstalling

Go to **Edit > Preferences > Add-ons**, expand **WeightLattice**, and click **Uninstall**.
