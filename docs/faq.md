# FAQ

## Can I use an existing vertex group?

Yes. WeightLattice works with the active vertex group, whether you created it manually or from the addon.

## Can I build a lattice from Edit Mode selection?

Yes. The addon includes a command that creates a lattice from selected vertices, edges, or faces.

## Can I go back from lattice editing to weight painting?

Yes. The workflow includes a direct return path from Edit Lattice back to Weight Paint.

## Can I remove the lattice without applying the deformation?

Yes. You can remove the lattice directly when you no longer need it.

## Can I edit the lattice resolution later?

Yes. The lattice properties panel lets you adjust resolution, interpolation, and modifier settings.

## Is the addon intended for technical artists?

Yes. It is built for practical deformation workflows and fits both artist-friendly and technical setups.
